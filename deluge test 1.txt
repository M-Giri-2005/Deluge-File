// 2. Check whether the given number is Armstong Number or Not. 
// Explanation: 
// An Armstrong number (also known as a narcissistic number) is a number 
// that is equal to the sum of its own digits, each raised to the power of the 
// number of digits. 
// Example: 153 : (1^3 + 5^3 + 3^3 = 1 + 125 + 27 =153)


input=153;
new_list=input.toList("");
size=new_list.size();
count=0;
for each ver in new_list
{
	sq=ver.power(size);
	count=count+sq;
}
if ( count == input ) 
{
	info "is a Armstong Number" + count ;
}
else
{
	info "is not a Armstong Number";
}
